# Plant Game

A small 3D Unity prototype where the player fights waves of enemies using a water based gun and movement system. The goal is to protect the main plant while simultaniously gathering enough water from the map to move, shoot and water the main plant.

## Features

- Movement System
- Weapon System
- Enemy-Spawn System
- Enemy-Behaviour System
- Plant State System
- Water-System for Plant, Weapons and Movement
- Roguelite Skill-Tree System
- Roguelite Resource System

## Installation

1. Clone the repository
2. Open the project in Unity 6
3. Press Play
